# NoClip Mod
Frustrated with trying to change something behind foliage?\
Want to move between parts of your build easier?\
NoClip allows you to clip through objects for an easier build experience!

### Info
- NoClip only works while flying in the Map Editor
- Toggle default state when starting the game
- Hotkey for quickly switching NoClip on/off (default: `N`)

## Install
This mod is used with [MelonLoader](https://github.com/LavaGang/MelonLoader).\
Mod dependency: [BBModMenu](https://github.com/MiaouZart/BBModMenu)\
You may also get this on [Thunderstore](https://thunderstore.io/c/beton-brutal/p/Beton_Bros/NoClip/).
